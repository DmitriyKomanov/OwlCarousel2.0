== Description ==

This is a slideshow built on the modern new Owl Carousel 2. Touch enabled jQuery plugin that lets you create a beautiful responsive carousel slider.
Enable it as a widget or display it as shortcode. This slideshow is meant to be easy to use for you so there is no complexity of adding it.
The slideshow is tested on: Chrome, Firefox, Opera, IE7/8/10/11, iPad Safari, iPod4 Safari, Nexus 7 Chrome, Galaxy S4, Nokia 8s Windows8 and minumum compatibile jQuery version is 1.8.3 version.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place shortcode [owl-shortcode] to display the slideshow. You can also enable it through the widget list

== Frequently Asked Questions ==

= How do I add a slideshow image? =

You can add a slideshow image through the WordPress admin area by clicking on Owl Carousel then add a title and featured image.

= How do I display the slideshow? =

You can display the slideshow through shortcode by [owl-shortcode] in text or by widget in the list.
